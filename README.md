# Happy Birthday, My Love 🎂💖

A romantic, interactive Single Page Application (SPA) designed to celebrate a special birthday. This project features a luxury aesthetic with smooth animations, floating heart effects, and a responsive memory gallery.

## ✨ Features

* **Typewriter Hero:** A smooth, cinematic typing animation for the birthday greeting.
* **Floating Hearts:** A dynamic background with floating emoji particles.
* **Polaroid Gallery:** A hover-interactive grid for "Our Beautiful Memories" with sliding captions.
* **Heartfelt Message:** A dedicated letter section with a focus on elegant typography.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Built With

* **HTML5** - Semantic structure.
* **CSS3** - Custom properties, Flexbox/Grid, and keyframe animations.
* **JavaScript (Vanilla)** - Intersection Observer API for scroll effects and dynamic DOM manipulation.

## 🚀 How to Use

1.  **Download** the project files (`index.html`, `style.css`, and `script.js`).
2.  **Add Photos:** Replace the placeholder image URLs in the `index.html` gallery section:
    ```html
    <img src="your-photo.jpg" alt="Memory">
    ```
3.  **Personalize:** Edit the text inside the `<div class="letter-content">` to include your own message.
4.  **Launch:** Open `index.html` in any modern web browser.

## 🎨 Customization

To change the color theme, update the CSS variables in `style.css`:
```css
:root {
    --bg-color: #fff0f3; /* Background */
    --accent: #ff4d6d;   /* Primary Romantic Color */
    --text-main: #590d22; /* Text Color */
}
